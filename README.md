## -Heart-Disease-Prediction

Модель искусственного интеллекта для выявления заболеваний сердца с использованием алгоритмов машинного обучения
# Аннотация
В статье описывается разработка системы для выявления заболеваний сердца с помощью искусственного интеллекта и алгоритмов машинного обучения. Показано, как машинное обучение позволяет прогнозировать риск развития сердечно-сосудистых заболеваний. Разработано приложение на Python для медицинских исследований. Рассмотрены этапы разработки: сбор данных, логистическая регрессия, оценка атрибутов. Для повышения точности диагностики реализован алгоритм классификатора случайного леса, достигший около 83% точности на обучающей выборке. Приведены результаты экспериментов, подтверждающие эффективность подхода.
# Введение
Сердечно-сосудистые заболевания — одна из главных причин смертности в мире. В здравоохранении накапливается большой объём данных, которые могут быть использованы для улучшения диагностики и лечения. Python позволяет эффективно обрабатывать эти данные, выявлять закономерности и строить прогнозные модели, что особенно важно для профилактики сердечно-сосудистых заболеваний.
# Цели и задачи исследования

# Обзор литературы
Рассмотрены существующие подходы к диагностике сердечно-сосудистых заболеваний с помощью машинного обучения. Описаны используемые библиотеки (Pandas, Numpy, Matplotlib и др.), этапы работы с данными (загрузка, обработка, анализ), а также различные алгоритмы классификации (логистическая регрессия, случайный лес, KNN и др.). Отмечено, что Python соответствует требованиям безопасности медицинских данных и позволяет строить эффективные модели прогнозирования.
# Подход к исследованию
Проект включает сбор данных из внешних источников, их обработку и анализ с помощью машинного обучения. Используются методы корреляционного анализа, классификации и оценки точности моделей. Особое внимание уделяется кибербезопасности и соответствию нормативам.
Авторы подчёркивают важность раннего выявления сердечно-сосудистых заболеваний для повышения эффективности лечения и снижения смертности.
# Основные этапы работы
•	Сбор и обработка данных: использован публичный датасет с медицинскими показателями пациентов (возраст, пол, уровень холестерина, давление и др.).
•	Предобработка данных: обработка категориальных переменных, заполнение пропусков, нормализация.
•	Применение алгоритмов машинного обучения: рассмотрены логистическая регрессия, K-ближайших соседей, дерево решений, случайный лес и SVM.
•	Выбор и обоснование модели: основным выбран алгоритм случайного леса (Random Forest) благодаря высокой точности и устойчивости к переобучению.
# Результаты
•	Точность модели: случайный лес показал точность около 83% на обучающих данных и около 70% на тестовых.
•	Сравнение с другими алгоритмами: Random Forest превзошёл по точности KNN, SVM и дерево решений.
•	Визуализация и анализ: использованы гистограммы, матрицы корреляций, heatmap для анализа взаимосвязей между признаками.
